# haruhishot

## thanks to wayshot

## Use example

Fullscreen (No, just the first screen, just because I am not familar with image.rs)

```
haruhishot | wl-copy
```

or

```
haruhishot >> suzumiya.png
```

Pick with Region

```
haruhishot -S (slurp -d) --stdout | wl-copy
```

or

```
haruhishot --slurp (slurp -d) --stdout | wl-copy
```

Get Lists

```
haruhishot -L
```

or
```
haruhishot --list
```
